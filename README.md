# 我钱呢？ (Where is my money?)

你的私人财务毒舌审计师。通过上传账单截图，AI 将为你提供最扎心的财务审计报告。

## 项目预览
- **前端**: Vue 3 + UnoCSS + Vite
- **后端**: Node.js + Express + Multer

## 核心功能
- **账单审计**: 上传微信/支付宝月度账单截图。
- **毒舌评价**: 基于财务状况的 AI 毒舌锐评。
- **贫穷倒计时**: 距离实现财富自由（或破产）的精确计算。
- **响应式设计**: 完美适配手机端和 PC 端。

## 快速开始

### 1. 后端启动
```bash
cd backend
npm install
node server.js
```

### 2. 前端启动
```bash
cd frontend-v3
npm install
npm run dev
```

## 部署说明
本项目包含 Node.js 后端，建议：
- **前端**: 部署至 Vercel, Netlify 或 GitHub Pages。
- **后端**: 部署至 Render, Railway 或 Heroku。
